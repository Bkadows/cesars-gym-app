🔧 Paso 2 — Si tienes la pestaña original abierta

Si la pestaña del gym sigue abierta en tu navegador, los datos pueden estar en memoria. Sigue estos pasos:

1
Abre la consola del navegador en la pestaña del gym
En Mac: Cmd + Opción + J · En Windows: F12 → Console
2
Pega este código para ver los datos de hoy en memoria: // Verificar ventas de hoy en memoria const hoy = new Date().toISOString().slice(0,10); const enMemoria = typeof FIREBASE_DATA !== 'undefined' ? FIREBASE_DATA.cg_ventas.filter(v => (v.date||v.payDate||'').startsWith(hoy)) : []; console.log('✅ Ventas de HOY en memoria:', enMemoria.length, enMemoria); enMemoria.length > 0 ? alert('✅ ENCONTRADAS: ' + enMemoria.length + ' ventas de hoy. Ve al paso 3.') : alert('⚠️ No hay ventas de hoy en memoria. Intenta IndexedDB.'); 👆 Haz clic para copiar
3
Si el paso 2 encontró datos, pega esto para restaurarlos en Firestore: // Restaurar ventas de hoy a Firestore (ejecutar solo si el paso 2 encontró datos) db.collection('store').doc('cg_ventas').set({ data: FIREBASE_DATA.cg_ventas }) .then(() => alert('✅ Datos restaurados en Firestore exitosamente!')) .catch(e => alert('❌ Error: ' + e.message)); 👆 Haz clic para copiar
4
Si el paso 2 NO encontró datos, pega esto para escanear IndexedDB: // Escanear IndexedDB de Firebase (async () => { const dbs = await indexedDB.databases(); console.log('IndexedDB disponibles:', dbs.map(d=>d.name)); const fbDb = dbs.find(d => d.name && d.name.includes('firestore')); if(!fbDb) { alert('No se encontró cache de Firebase en IndexedDB'); return; } const req = indexedDB.open(fbDb.name, fbDb.version); req.onsuccess = e => { const db2 = e.target.result; console.log('Stores:', [...db2.objectStoreNames]); const tx = db2.transaction(['remoteDocumentsV14','remoteDocuments'].filter(s=>db2.objectStoreNames.contains(s)), 'readonly'); const stores = [...db2.objectStoreNames]; console.log('Object stores:', stores); alert('Cache encontrado: ' + fbDb.name + '\nStores: ' + stores.join(', ') + '\nRevisa la consola para más info.'); }; })(); 👆 Haz clic para copiar
